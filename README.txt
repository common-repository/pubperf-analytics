== Description ==

[Pubperf Analytics](https://www.pubperf.com/) plugin provides all in one performance statistics insight about your website: website speed, [Prebid.js and header bidding analytics](https://www.pubperf.com/prebid-analytics), advertising analytics, user experience monitoring, website uptime monitoring and checking.

Features:

* Realtime Prebid.js and Header bidding Analytics
* Lighthouse Performance Snapshot
* Real User Performance Monitoring
* Ads Loading Speed and Total Request Count
* Website Speed Index and Memory Usage
* Header Bidding Percentage and Bidders Monitoring
* More

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `pubperf` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the Pubperf admin page and configure your settings.
4. Go to the Settings > Pubperf, enter your [Pubperf Website ID aka license ID](https://www.pubperf.com/), and click save.
